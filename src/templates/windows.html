Recent ZTClient versions (1.34.0 and higher)

ZTClient added Fast User Switching in version 1.34 and you can now use the new login command to connect to one or more ninjapanda (and ZTClient) servers. The previously used profiles does not have an effect anymore.

Use ZTClient's login command to add your profile:

ZTClient login --login-server {{.URL}}

Windows registry configuration (1.32.0 and lower)

This page provides Windows registry information for the official Windows ZTClient client.

The registry file will configure ZTClient to use {{.URL}} as its control server.

Caution

You should always download and inspect the registry file before installing it: 

curl {{.URL}}/windows/ZTClient.reg

Installation

Ninjapanda can be set to the default server by running the registry file:

Windows registry file

  1. Download the registry file, then run it
  2. Follow the prompts
  3. Install and run the official windows ZTClient client
  4. When the installation has finished, start ZTClient, and log in by clicking the icon in the system tray

Or

Open command prompt with Administrator rights. Issue the following commands to add the required registry entries: 

      REG ADD "HKLM\Software\ZTClient IPN" /v UnattendedMode /t REG_SZ /d always
            REG ADD "HKLM\Software\ZTClient IPN" /v LoginURL /t REG_SZ /d "{{.URL}}"

Restart ZTClient and log in.
